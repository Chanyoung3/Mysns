# MYSNS

---

## 프로젝트 설명  

  - JAVA와 JSP를 기반으로 개발한 나만의 SNS
  - **개발 기간**: 2023년 3월 ~ 2023년 6월  

---

## 주요 기능

- 게시글 등록 및 삭제

- 친구 추천, 추가 및 삭제
